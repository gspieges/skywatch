How to use SkyWatch
1. Find a flight
Click any aircraft on the map. Note its callsign in the popup.
2. Enter callsign
Type it in the box at the top and click SET ATC.
3. Auto mode
When the flight is found, phase detects automatically and ATC updates every 20 seconds.
4. Manual mode
If not found, select the phase yourself. Green listen buttons appear immediately.
5. Audio
Click the green Listen button to open the correct LiveATC search page. Pick a feed that is online and press play.
